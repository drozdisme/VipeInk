## Фокус на сейчас
1. Зафиксировать состав reusable core MVP
2. Определить smallest convincing demo
3. Разложить build на 7 дней / 14 дней / 30 дней
4. Подготовить grant-ready и contest-ready упаковку
5. Выбрать первые 2-3 внешние opportunity targets

## Что именно фиксируем первым
- один end-to-end сценарий settlement
- минимальный API и state flow
- audit trail / visibility слой без лишней сложности
- demo console или другой простой интерфейс показа

## Чего не делаем сейчас
- полный rollup scope
- отдельный consumer product
- широкий bank enterprise scope без demo core
